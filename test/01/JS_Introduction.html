JavaScript Can Change HTML Styles (CSS)

definir el parragraph: <p id="demo">
cambiar el attribute: document.getElementById('demo2').style.fontSize

JavaScript can change the style of an HTML element.

JavaScript Can Hide HTML Elements

definir el paragraph: <p id="demo3">
cambiar el attribute: document.getElementById('demo3').style.display='none'

JavaScript can hide HTML elements.

JavaScript Can Show HTML Elements
